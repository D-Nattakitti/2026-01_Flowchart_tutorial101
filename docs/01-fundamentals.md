0 - ความสำคัญของ Flowchart
    อืมมม ถ้าให้พูดว่ามันสำคัญแค่ไหนผมก็คงต้องเรียนตามตรงว่ามันค่องข้างที่จะสำคัญมากในส่วนหนึ่ง อยากให้คุณมองการเขียนโปรแกรมเป็นการเดินทาง การมี flowchart จะช่วยให้คุณมองภาพรวมได้ง่ายขึ้นก็เหมือน คุณจะมีแผนที่ในการเดินทางมันเลยทำให้เราเดินทางได้สะดวกขึ้น ถ้าถามว่าการทำงานโดยไม่มี Flowchart หรือการพัฒนาแบบ MVP เลยจะไม่ไวกว่าหรอ? สำหรับผมแล้วใช่ครับ มันไวกว่า แต่มันก็ทำให้เราทำเพิ่มไปเรื่อยๆ แล้วกว่าจะรู้ตัวอีกที flowchart ระบบเล็กๆที่เราต้องการก็ใหญ่เกิน และบางครั้งเราก็อาจสับสนมันได้ เพราะงั้นการวางแผนในเรื่อง Flowchart ก่อนก็ถือเป็นเรื่องที่ดีครับ
